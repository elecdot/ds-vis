    PLAN — Iteration Roadmap

    本文件是“路线图与里程碑索引”，用于组织未来迭代的工作包与交付物清单。

    约束： - 不重复当前状态：当前阶段与完成情况以 docs/project_state.md 为准（SSOT）。 - 这里只记录“计划做什么/交付什么”，以及对应的文档与代码绑定点（方便协作拆分）。

    1. Roadmap（High-level）

    P0.7（基线巩固）

    • 目标：List + Tree baseline 可复用范式、端到端链路稳定（Model→Layout→SceneGraph→Renderer/UI→Persistence/DSL）。
    • 交付：以 docs/project_state.md 为准。

    P0.8（体验与架构补强）

    • 动画：非阻塞调度 / skip/seek（至少提供“加速+跳步”能力）。
    • 布局：布局引擎去状态化（或提供可重建接口），树/DAG 重叠治理。
    • 展示：补齐“课程展示脚本 + 图示证据链”（Demo + Diagrams）。

    P0.9 / v0.2（扩展）

    • AVL / 红黑树：旋转微步骤范式、可复用的树重连动画。
    • DSL：变量/作用域/脚本化；LLM 安全接入（可选）。
    • Git：lane 布局增强、标签与 merge 表达。

    2. Deliverables Checklist（建议）

    • Docs
    • docs/demo/scenarios.md：演示脚本（DSL/JSON）。
    • docs/diagrams/index.md：架构/流程图 + 可复现生成方式。
    • docs/design/layout_v2.md：布局架构重构的设计绑定（如进入 P0.8）。
    • Code
    • Layout 引擎：重建/无状态计算接口（为 seek/重放铺路）。
    • Renderer：播放控制（暂停/步进/加速）与基本 skip。
    • Tests：layout/renderer 的属性断言覆盖（避免截图测试）。

    入口导航： docs/index.md（Registry） / docs/project_state.md（SSOT）
